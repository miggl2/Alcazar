# AI 일반 상식 체크리스트

15줄 이하로 유지한다. 작업 시작 전 훑는다.

- 사용자 입력은 절대 신뢰하지 말고 길이 제한·형식 검증·서버 검증을 둔다.
- 에러는 한국어 메시지로 바꾸고 스택트레이스는 사용자에게 보이지 않는다.
- 모든 화면에는 빈 상태, 로딩, 에러 상태를 준비한다.
- 모바일은 `100vh` 대신 `100dvh`, 노치는 safe-area를 적용한다.
- 한글 줄바꿈은 `word-break: keep-all`, 긴 URL만 별도 줄바꿈 처리한다.
- 게임 세이브는 스키마 버전과 마이그레이션을 둔다.
- 점수·결제·권한은 클라이언트 값을 믿지 말고 서버에서 재검증한다.
- 시크릿은 `.env`에만 두고 `NEXT_PUBLIC_` 접두사를 붙이지 않는다.
- 새 패키지는 `npm view` 확인 후 `~` 버전으로 고정한다.
- 5개 초과 파일은 작업을 쪼개고 사용자 확인을 받는다.
- 완료 전 `npm run check`와 직접 확인 방법 안내를 끝낸다.
