# Google Play 스토어 등록 초안

Play Console의 Store listing 화면에 옮겨 적기 위한 초안이다. 실제 제출 전에는 스크린샷과 그래픽을 최종 이미지로 교체한다.

`npm run play:ready`는 아래 Play Console 제한을 검사한다.

- 앱 이름: 30자 이하
- 짧은 설명: 80자 이하
- 긴 설명: 4000자 이하

## 기본 정보

| 항목 | 초안 |
|---|---|
| 앱 이름 | Alcazar |
| 기본 언어 | 한국어 |
| 앱/게임 | 게임 |
| 카테고리 | 퍼즐 |
| 가격 | 무료 |
| 광고 포함 여부 | 아니오 |
| 앱 내 구매 | 없음 |
| 연락 이메일 | TODO_CONTACT_EMAIL |
| 개인정보 처리방침 | `https://실제-도메인/privacy-policy.html` |

## 짧은 설명

```text
격자의 모든 칸을 한 번씩 지나는 길을 찾는 퍼즐 게임.
```

## 긴 설명

```text
Alcazar는 격자 안의 모든 칸을 한 번씩 지나도록 길을 완성하는 퍼즐 게임입니다.

입구가 있는 문제, 여러 입구가 있는 문제, 입구 없는 순환형 문제를 지원합니다.
시드 기반 생성으로 같은 조건의 퍼즐을 다시 만들 수 있고, 기록은 기기 안에 저장됩니다.

광고, 로그인, 결제 없이 플레이할 수 있습니다.
```

## 이미지 자료 체크리스트

최종 파일 경로와 촬영 기준은 `docs/PLAY_STORE_ASSETS.md`에서 관리한다.

| 자료 | 필요 상태 |
|---|---|
| 앱 아이콘 | 512x512 PNG 필요 |
| 피처 그래픽 | 1024x500 PNG 또는 JPG 필요 |
| 휴대폰 스크린샷 | 2장 이상 필요 |
| 7인치 태블릿 스크린샷 | 선택 |
| 10인치 태블릿 스크린샷 | 선택 |

## 추천 스크린샷 구성

1. 메인 메뉴 화면
2. 퍼즐 생성 패널이 열린 화면
3. 실제 퍼즐 플레이 화면
4. 클리어 팝업 화면
